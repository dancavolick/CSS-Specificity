CSS - Specificity

This table details the Specificity Rules engaged within Cascading Style Sheets (CSS)

Specificity is the means by which browsers decide which CSS property values are the most relevant to an element and, therefore, will be applied. Specificity is based on the matching rules which are composed of different sorts of CSS selectors

Score Type Examples Description Notes
1000 Inline Styles Keyword "style" followed by the CSS rule within the opening tag of an element within the HTML document These appear inline within the HTML document itself and dominate any contrasting rule, for any Selector type, held on an external Stylesheet n/a
0100 ID Selectors #example, id="example" These should appear only once in the HTML document These rules dominate contrasting Class and Element rules
0010 Attribute/Class/Pseudo-Class Selectors .example, class="example" These can appear multiple times in the HTML document These rules dominate those applied to Elements/Pseudo Elements
0001 Element/Pseudo Element Selectors Elements/Pseudo elements: e.g. h1/:hover These can appear multiple times in the HTML document and hold the lowest score These rules do not dominate rules applied to any other Selectors

Key considerations

  • Equal Scoring: If a Selector has been listed in a mutliple Selector ruleset more than once and the rules contrast, the ruleset with the highest score will dominate, for example:
    Multiple ruleset one: #id.class p {color="red";}.........Score #id(100) + .class(10) + element(1) = 111
    Multiple ruleset two: #id p {color="blue";}..........Score #id(100) + element(1) = 101
    Multiple ruleset one will dominate as it calculates to the highest score: 111

  • Duplication: If there are contrasting rules that have been set at the same level, for example, a developer has inadvertently set 2 different background-colors for the Paragraph Selector, the rule listed furthest down in the Stylesheet dominates

  • !important: If the !important rule has been applied to any Selector, that rule will dominate any contrasting rules
